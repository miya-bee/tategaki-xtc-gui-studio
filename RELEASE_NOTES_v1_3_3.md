# tategakiXTC GUI Studio v1.3.3 Release Notes

この文書は、縦書きXTC GUI Studio v1.3.3 の更新内容をまとめたものです。

- 公開向けバージョン表記: `v1.3.3`
- 基準: `v1.3.2`
- 公開対象: Python GUI版のみ

## 主な更新

v1.3.3 は、v1.3.2 公開後の hotfix 版です。公開説明の整合性を直し、CJK 互換縦書き閉じ括弧を直接入力した場合の下鍵括弧補正漏れを修正しました。

- README の `v1.3.2 の主な更新` / `v1.3.1 の主な更新` と、前回公開版表記の不一致を修正しました。
- CJK 互換縦書き閉じ括弧 `﹂` / `﹄` を直接入力した場合にも、下鍵括弧の5モード補正が適用されるようにしました。
- `﹂` / `﹄` 直接入力時の下補正・上補正を確認する回帰テストを追加しました。

## GitHub Release

- Release tag: `v1.3.3`
- Release title: `v1.3.3`
- 添付ファイル: `tategaki-xtc-gui-studio_v1.3.3-release.zip`

## 確認ポイント

- アプリ左上のバージョン表示が `1.3.3` になっていること。
- README 上部の前回公開版が `v1.3.2`、今回公開版が `v1.3.3` になっていること。
- README に `v1.3.3` / `v1.3.2` / `v1.3.1` の主な更新セクションが順に存在すること。
- 下鍵括弧の `下補正 弱` / `下補正 強` で、閉じ鍵括弧 `」` / `』` / `﹂` / `﹄` が標準より下へ寄ること。
- プレビュー枚数の初期値が `20` のままであること。
