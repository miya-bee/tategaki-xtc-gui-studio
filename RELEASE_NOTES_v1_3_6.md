# TategakiXTC GUI Studio v1.3.6

v1.3.6 は、v1.3.5 からの差分をまとめた GitHub 公開版です。

ページ番号表示、EPUB 入力の堅牢化、プレビュー更新制御、ファイルビューワーモード改善、保存先処理改善、位置補正機能追加、XTC / XTCH 書き出し信頼性強化を含みます。

内部作業版としては v1.3.6.45 相当を基準にしていますが、公開版の表記は v1.3.6 に統一しています。

## 主な更新

### ページ番号表示

- 出力ページ右下に `1/100` のようなページ番号を表示できるようにしました。
- 初期状態は OFF です。
- ページ番号フォントサイズを指定できます。
- ページ番号 ON 時は、下余白が自動的に不足しないよう補正されます。

### EPUB 入力の堅牢化

- EPUB spine の `linear="no"` を適切にスキップします。
- 複雑な ruby / rtc 構造でルビ文字が本文に漏れにくくなりました。
- CSS の子孫セレクタ / 子セレクタの扱いを改善し、`display:none` などの過剰適用を抑制しました。
- EPUB 内画像の検出対象を拡張しました。
- 大きな EPUB の一部経路で、ページ保持によるメモリ負荷を軽減しました。

### プレビューとファイルビューワーモード

- 起動直後にプレビュー表示が「生成中…」のまま残る問題を修正しました。
- プレビュー更新ボタンの状態同期を強化しました。
- XTC / XTCH ファイルビューワーモードから通常入力へ戻る経路を改善しました。
- ファイルビューワーモード中にカーソルキー操作で通常サンプルへ戻ってしまう問題を修正しました。

### 保存先処理

- 単体変換で別保存先を指定した場合、変換結果と「保存先を開く」が指定フォルダを指すよう改善しました。
- 「保存先リセット」後は、ソースファイル側フォルダへ戻るよう修正しました。
- 保存先指定の検証を整理し、引用符付きパスや異常系の扱いを強化しました。

### 位置補正機能

- 半角数字/記号、縦中横記号、半角英字などの位置補正を追加・改善しました。
- 半角英字は A-Z / a-z のみを対象にし、全角英字やルビ内英字には影響しません。
- 位置補正確認用の `sample_texts/` を同梱しました。

### XTC / XTCH 書き出し信頼性

- TXT / Markdown / EPUB などの主経路で、XTC / XTCH temp file の `flush()` / `fsync()` と `os.replace()` 前の自己検証を追加しました。
- フォルダ一括変換のキャンセル判定を型判定へ変更し、通常エラーメッセージによる誤停止を避けるようにしました。
- 連番保存先候補の探索に上限を設けました。

## 配布内容

GitHub Release では Python GUI 版の source 配布 zip を公開します。

- `tategaki-xtc-gui-studio_v1.3.6-release.zip`
- `tategaki-xtc-gui-studio_v1.3.6-release.zip.sha256.txt`

source 配布版の実行には Python 環境が必要です。

Windows exe / portable 版は GitHub には置かず、note 有料配布向けに別途配布します。

## 実機確認ポイント

1. 起動直後に「生成中…」表示が残らない。
2. TXT / Markdown / EPUB を開いてプレビューできる。
3. XTC / XTCH 保存ができる。
4. 別保存先指定と「保存先を開く」が正しく動く。
5. 「保存先リセット」後、ソースファイル側フォルダへ戻る。
6. ファイルビューワーモードから通常入力へ戻れる。
7. sample_texts の位置補正確認用ファイルが同梱されている。
