半角英字位置補正テスト v1

このテキストは、半角英字の上下位置補正を確認するためのサンプルです。
半角英字は、これまで通り1文字ずつ縦に並びます。
今回の確認対象は、A-Z / a-z の上下位置だけです。

確認語:
XTC
XTCH
EPUB
PDF
GitHub
Windows
Python
AlphaBeta
A-12
X4
v1.3.6

「半角英字」の位置補正を、標準、下補正弱、下補正強、上補正弱、上補正強に切り替え、
XTC、EPUB、PDF、GitHub、Windows の英字だけが上下に動くか確認してください。

半角数字/記号との切り分け確認:
A-12 / X4 / v1.3.6 / EPUB3 / PDF-2026
数字、ハイフン、ピリオド、スラッシュ、パーセントなどは「半角数字/記号」側の補正対象です。
半角英字は「半角英字」側で動き、数字や記号は一緒には動かない想定です。

ルビ内英字は対象外です。
｜英字《ABC》を含むルビでは、ルビ側のABCは半角英字補正で動かない想定です。
