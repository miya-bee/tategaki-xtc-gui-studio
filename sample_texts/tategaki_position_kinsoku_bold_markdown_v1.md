# 縦書きXTC GUI Studio 位置補正・禁則・太字確認用 Markdown v1

この Markdown 版は、**太字が実際に反映されるか** を確認するためのテスト用テキストです。
TXT版では青空文庫風の太字注記は未対応なので、太字確認はこちらを使ってください。

確認対象は、**句読点**、**漢数字「一」**、**半角数字/記号**、**縦中横記号**、**下鍵括弧**、**禁則処理**、**ルビ**、**太字**です。

## 確認ページ 01：句読点と太字

**「これは太字の会話文です。」** と、彼はゆっくり言った。
通常本文では「句点が右に寄りすぎていないか、読点が下がりすぎていないか」を確認します。
**太字の中で、読点、句点、疑問符？、感嘆符！、閉じ括弧」が続く場合**、位置が崩れないかを見てください。

**共通キャリブレーション：** 、。！？！！？？？！　!! !? ?! ??　一 第一 一日 一九二六年　1 12 88 123 2026 12.5 3.14 10% 1/100 2026/05/22　「終わりです。」『終わりです。』﹁終わりです﹂﹃終わりです﹄。
