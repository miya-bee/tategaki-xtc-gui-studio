半角英字・全角英字 比較テスト v1

このテキストは、半角英字と全角英字の見え方を比較するためのサンプルです。
「半角英字」位置補正は A-Z / a-z の半角英字だけに効きます。
全角英字（Ａ-Ｚ / ａ-ｚ）は、半角英字補正では動かない想定です。

ＡｌｐｈａＢｅｔａ

混在確認:
半角 XTC と全角 ＸＴＣ を同じ行で比較します。
半角 EPUB と全角 ＥＰＵＢ を同じ行で比較します。
半角 GitHub と全角 ＧｉｔＨｕｂ を同じ行で比較します。
半角 Windows と全角 Ｗｉｎｄｏｗｓ を同じ行で比較します。

数字・記号との切り分け:
X4 と Ｘ４
A-12 と Ａ－１２
v1.3.6 と ｖ１．３．６
PDF-2026 と ＰＤＦ－２０２６

確認方法:
1. 「半角英字」を標準から下補正/上補正へ切り替えます。
2. 半角英字だけが上下に動くことを確認します。
3. 全角英字は半角英字補正では動かないことを確認します。
4. 数字や記号は「半角数字/記号」側の補正対象として残ることを確認します。
