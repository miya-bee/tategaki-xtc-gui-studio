# tategakiXTC GUI Studio v1.3.2 Release Notes

この文書は、縦書きXTC GUI Studio v1.3.2 の更新内容をまとめたものです。

- 公開向けバージョン表記: `v1.3.2`
- 基準: `v1.3.1`
- 公開対象: Python GUI版のみ

## 主な更新

v1.3.2 は、v1.3.1 のフォルダ一括変換・サブフォルダ処理・既存ファイル扱い・optional dependency 整理を引き継ぎつつ、下鍵括弧の位置補正とプレビュー初期値を調整した小改良版です。

- 下鍵括弧の補正モードを、句読点・漢数字一と同じ5モードへ拡張しました。
  - `下補正 強`
  - `下補正 弱`
  - `標準`
  - `上補正 弱`
  - `上補正 強`
- 下鍵括弧の `下補正` が、プレビューと実変換の描画処理で実際に反映されるようにしました。
- プレビュー枚数の初期値を 10 ページから 20 ページに変更しました。

## 確認ポイント

- アプリ左上のバージョン表示が `1.3.2` になっていること。
- プレビュー枚数の初期値が `20` になっていること。
- 下鍵括弧の補正モードに5項目が表示されること。
- 下鍵括弧の `下補正 弱` / `下補正 強` で、閉じ鍵括弧 `」` / `』` / `﹂` / `﹄` が標準より下へ寄ること。
