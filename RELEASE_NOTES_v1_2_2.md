# tategakiXTC GUI Studio v1.2.2 Release Notes

- 公開向けバージョン表記: `v1.2.2`
- 基準版: `v1.2.1`
- 公開対象: Python GUI版
- 種別: 安定版

## 概要

v1.2.2 は、v1.2.1 を基準に、波ダッシュ・チルダ類の縦書き表示調整を追加した安定版です。

主に、次のような文字の縦書き表示を確認・調整しやすくしました。

```text
～ 〜 〰 ~ ∼ ∽ ∿ ≀
```

## 主な変更点

### 1. 波線描画設定を追加

波ダッシュ・チルダ類について、描画方式を選択できるようにしました。

- 回転グリフ
- 別描画

初期値は `回転グリフ` です。

### 2. 波線位置設定を追加

波線類の描画位置を選択できるようにしました。

- 標準
- 下補正弱
- 下補正強

初期値は `標準` です。

### 3. GUI変更時の即時プレビュー反映

波線描画・波線位置を変更したとき、プレビューへ即時反映されるようにしました。

### 4. 設定保存・復元に対応

波線描画・波線位置は、ini 保存・読み込みに対応しています。

アプリを再起動しても、前回の設定が復元されます。

### 5. 変換処理への反映

波線描画・波線位置は、preview payload と実変換用 conversion args に反映されます。

プレビューと変換結果で設定がずれないようにしました。

### 6. 表記ゆれ正規化

GUI表示、保存値、内部値の表記ゆれを正規化し、不正値が入った場合も安全側へフォールバックするようにしました。

### 7. プリセット仕様表示の整理

左ペイン上部のプリセット仕様表示は、基本仕様に絞る方針としました。

波線描画・波線位置はフォント依存のチューニング項目であるため、プリセット仕様表示からは外しています。

設定項目自体はGUI上に残っており、保存・復元・プレビュー反映・変換反映も維持しています。

## 既知の注意点

- 波線類の見え方はフォントに依存します。
- 半角コンマ `,` や半角ピリオド `.` は、横書き前提の記号であり、縦書き表示では不自然に見える場合があります。
- README や Markdown のような横書き前提文書は、縦書き化したときに記号や英数字が不自然に見える場合があります。
- v1.2.2 では、青空文庫系の小説本文を主対象としているため、半角句読点の自動置換は行っていません。

詳しくは `KNOWN_LIMITATIONS.md` を参照してください。
