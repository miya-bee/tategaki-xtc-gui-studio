# TategakiXTC GUI Studio v1.5.0 Public Release Notes

v1.5.0 Public は、v1.4.2 以降の v1.5 開発系列で追加・修正した内容を、GitHub公開用のPublic版として整理した版です。
青空文庫の検索・ダウンロード機能とクリップボード変換機能は削除し、通常ファイル入力、フォルダ一括変換、EPUB / Markdown / TXT / 画像入力、XTC / XTCH 出力を中心に整えています。


## Public版とFull版

このGitHub配布版は `tategaki-xtc-gui-studio_v1.5.0-public.zip` として公開します。有償配布中のFull版は `tategaki-xtc-gui-studio_v1.5.0-full.zip` として扱います。Full版には、Public版では削除した青空文庫ダウンロード機能とクリップボード変換機能が含まれます。

Full版は note にて有償配布中です。

- 有償配布版: https://note.com/miya_bee_note/n/n8e8424e96e4e

## 主な変更点

- 青空文庫の検索・ダウンロード機能を削除しました。上部バーの「青空文庫」ボタン、検索サブウィンドウ、HTML/TXT取得処理、取得用キャッシュ処理は公開版から外しています。
- クリップボード変換機能を削除しました。上部バーの「クリップボード」ボタンと、コピー済みテキストを一時TXTとして読み込む処理は公開版から外しています。
- 欧文表示に「横組み」を追加しました。草枕など縦書き本文中の英文を、必要に応じて横方向のまとまりとして扱えます。
- URL を横組み対象に追加しました。長い URL は `/` / `.` / `-` / `_` / `?` / `&` / `=` / `#` などの区切りで分割します。
- 不完全な URL 風文字列、特に `https://` / `http://` / `www.` 単体を URL 横組みとして扱わないよう修正しました。
- 「行頭鍵括弧」設定を追加しました。従来どおりの「空白なし」を既定にしつつ、岩波文庫風の「1文字下げ」も選べます。
- 複数空行、会話行、ぶら下がり句読点直後の空行、短いルビ＋読点など、草枕テキストで確認した縦組み上の不自然さを修正しました。
- 右ペインの `XTCファイルを開く` / `PNG保存` まわりの表示を整理し、上部・中央ペインの横幅も調整しました。

## 変更していないもの

- ローカルTXTの青空文庫記法パーサ、ルビ・外字・傍点・傍線・改ページ・字下げ注記の処理は維持しています。
- EPUB / Markdown / ZIP / CBZ / 画像入力、フォルダ一括変換、XTC / XTCH 出力、ページ番号、進捗バー、プリセット、設定保存形式は維持しています。
- 欧文横組みを使わない場合の既存縦組み描画は、従来仕様を維持します。

## 確認ポイント

- 上部バーから「青空文庫」「クリップボード」ボタンが消えていること。
- 上部ボタンのヘルプから、青空文庫検索・クリップボード入力の説明が消えていること。
- 通常のTXT / Markdown / EPUB / 画像 / フォルダ一括変換が従来どおり動くこと。
- ローカルの青空文庫形式TXTで、ルビ・外字・傍点・傍線・空行が従来どおり変換できること。
- `https://example.com` / `www.example.com/path` / `example.com/foo_bar#top` が横組み対象になること。
- `https://` / `http://` / `www.` 単体が横組み対象にならないこと。
- 草枕TXTの英文、URL、空行、会話行、短いルビ＋読点が意図どおり表示されること。

詳細な変更履歴は `CHANGELOG.md` を参照してください。
