# tategakiXTC GUI Studio v1.3.1 Release Notes

この文書は、縦書きXTC GUI Studio v1.3.1 の更新内容をまとめたものです。

- 公開向けバージョン表記: `v1.3.1`
- 前回公開版: `v1.3.0`
- 公開対象: Python GUI版

## 位置づけ

v1.3.1 は、v1.3.0 で追加したフォルダ一括変換を、実際の運用で分かりやすく使えるように整えた小改良版です。
変換ロジック本体の大きな変更ではなく、完了表示、失敗時の案内、進捗表示、停止時の文言、上ペインのボタン説明を中心に整理しています。

## 主な更新

- フォルダ一括変換完了後に「出力フォルダを開く」導線を追加しました。
- 失敗ファイルがある場合の完了ダイアログとログ表示を改善しました。
- ログ末尾に `[ERROR-SUMMARY]` を出し、失敗内容を追いやすくしました。
- EPUB optional dependency 不足時の案内を改善し、`install_requirements.bat` / `requirements.txt` の利用を案内するようにしました。
- フォルダ一括変換中の進捗表示を `1/N 件目｜ファイル名` 形式に整理しました。
- v1.3.0 で追加したサブフォルダ処理、フォルダ構造保持、既存ファイルのスキップ / 別名保存 / 上書きの導線は維持しています。
- 停止ボタン押下後の文言を自然化しました。
- 停止完了時に、成功・スキップ・失敗・処理済み件数に加えて未処理件数を表示するようにしました。
- 停止時の `[STOP]` ログにも未処理件数を記録するようにしました。
- 上ペインのボタン名を「ファイルを開く...」「保存先を選ぶ...」「フォルダ一括変換...」に整理し、3つの違いを説明する `?` ヘルプを追加しました。

## 注意

EPUB 変換には optional dependency が必要です。不足している場合は、表示される案内に従って `requirements.txt` を入れ直すか、必要な追加ライブラリをインストールしてください。

フォルダ一括変換を停止した場合、現在処理中のファイルが終わってから停止します。そのため、停止ボタンを押した直後に処理が即時中断されない場合があります。
