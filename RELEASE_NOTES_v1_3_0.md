# tategakiXTC GUI Studio v1.3.0 Release Notes

この文書は、縦書きXTC GUI Studio v1.3.0 の更新内容をまとめたものです。

- 公開向けバージョン表記: `v1.3.0`
- 前回公開版: `v1.2.2`
- 公開対象: Python GUI版

## 主な更新

- フォルダ一括変換を追加しました。
- サブフォルダ内の対象ファイル検索に対応しました。
- フォルダ構造を保持した出力に対応しました。
- 既存ファイルがある場合の扱いとして、スキップ、別名保存、上書きを選べるようにしました。
- 完了表示とログに、成功・スキップ・失敗・処理済み件数・出力先・スキップ内訳を表示するようにしました。
- EPUB optional dependency 不足時の案内を改善しました。
- 画像由来の変換結果を正しい XTC / XTCH コンテナとして保存するようにしました。

## 注意

EPUB 変換には optional dependency が必要です。不足している場合は、表示される案内に従って追加ライブラリをインストールしてください。

v1.2.2.25 以前の開発版で作成した画像由来 `.xtc` は、v1.2.2.26 以降で再変換してください。


## 公開ファイル

- Release tag: `v1.3.0`
- Release title: `v1.3.0`
- 添付ファイル: `tategaki-xtc-gui-studio_v1.3.0-release.zip`

## 実機確認

RC1 の Windows 実機最終受け入れテストで、TXT 単体変換、フォルダ一括変換、既存ファイルスキップ、サブフォルダ構造保持、PNG 由来 `.xtc` 読み込みを確認済みです。
